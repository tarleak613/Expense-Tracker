# Privacy Policy

Last Updated: June 2026

Expense Inbox respects user privacy.

## Information Collected

The application may access transaction-related notifications displayed on the user's Android device when Notification Access permission is granted.

Examples include:

* Payment notifications
* Banking transaction notifications
* UPI transaction notifications

## Data Storage

All data is stored locally on the user's device.

The application does not upload, transmit, or share notification data with any external server.

## Third-Party Services

Expense Inbox does not use analytics, advertising, or tracking services.

## Data Sharing

The application does not sell, share, or disclose user information to third parties.

## User Control

Users may:

* Revoke Notification Access at any time
* Clear application data through Android settings
* Uninstall the application

## Contact

For questions regarding this Privacy Policy, please open an issue on the GitHub repository.
